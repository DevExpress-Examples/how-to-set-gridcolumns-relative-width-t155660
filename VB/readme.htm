How to set GridColumn?s relative width

To set GridColumn?s relative width, you can create behavior that contains the attached WidthMultiplier property. This property has a multiplier for calculation and its default value equals 1. If GridColumn?s width and WidthMultiplier are not set, the available width will be divided equally between all these columns.
